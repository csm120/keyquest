Press NVDA N to open the NVDA menu.
Press NVDA Q to quit NVDA.
Press NVDA Control C to save configuration.
Press NVDA Control R to reload configuration.
Press NVDA F12 to hear the time or date.
Press NVDA T to hear the window title.
Press NVDA Tab to hear the current focus.
Press NVDA B to read the active window.
Press NVDA C to read the clipboard text.
Press NVDA Shift B to hear the battery status.
Press NVDA Down Arrow to say all from the current position.
Press NVDA Up Arrow to read the current line.
Press NVDA Shift Up Arrow to read the selected text.
Press NVDA F to get text details like font and style.
Press NVDA F2 to pass the next key through to the application.
Press NVDA S to cycle speech modes.
Press NVDA 1 to toggle input help.
On a web page, press NVDA F7 to open the Elements List.
On a web page, press NVDA Control F to find text.
On a web page, press NVDA F3 to find the next match.
On a web page, press NVDA Shift F3 to find the previous match.
On a web page, press H to jump to the next heading.
On a web page, press Shift H to jump to the previous heading.
On a web page, press 1 through 6 to jump to a heading level.
On a web page, press K to jump to the next link.
On a web page, press Shift K to jump to the previous link.
On a web page, press E to jump to the next edit box.
On a web page, press Shift E to jump to the previous edit box.
On a web page, press F to jump to the next form field.
On a web page, press Shift F to jump to the previous form field.
On a web page, press B to jump to the next button.
On a web page, press Shift B to jump to the previous button.
On a web page, press D to jump to the next landmark.
On a web page, press Shift D to jump to the previous landmark.
On a web page, press T to jump to the next table.
On a web page, press Shift T to jump to the previous table.
On a web page, press L to jump to the next list.
On a web page, press Shift L to jump to the previous list.
On a web page, press X to jump to the next checkbox.
On a web page, press Shift X to jump to the previous checkbox.
On a web page, press C to jump to the next combo box.
On a web page, press Shift C to jump to the previous combo box.
On a web page, press R to jump to the next radio button.
On a web page, press Shift R to jump to the previous radio button.
On a web page, press U to jump to the next unvisited link.
On a web page, press Shift U to jump to the previous unvisited link.
On a web page, press V to jump to the next visited link.
On a web page, press Shift V to jump to the previous visited link.
On a web page, press O to jump to the next object, such as a web dialog box.
On a web page, press Shift O to jump to the previous object.
