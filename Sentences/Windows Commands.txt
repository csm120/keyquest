Press Windows key to open Start menu.
Press Alt Tab to switch between open windows.
Press Windows D to show or hide desktop.
Press Windows E to open File Explorer.
Press Windows L to lock your computer.
Press Windows N to open Notification Center and calendar.
Press Windows I to open Settings.
Press Windows S to open Search.
Press Windows V to open the clipboard or clipboard history.
Press Windows X to open Quick Link menu.
Press Windows Period to open emoji picker.
Press Control C to copy selected text.
Press Control X to cut selected text.
Press Control V to paste copied text.
Press Control Z to undo last action.
Press Control Y to redo last action.
Press Control A to select all items.
Press Control F to find text.
Press Control P to print current document.
Press Control S to save current document.
Press Alt F4 to close current window.
Press Alt Enter to open Properties for the selected item.
Press Alt Space to open window menu.
Press F2 to rename selected file.
Press F5 to refresh current view.
Press Escape to cancel current operation.
Press Delete to move item to Recycle Bin.
Press Shift Delete to permanently delete item.
Press Control Shift Escape to open Task Manager.
Press Windows R to open Run dialog.
Press Windows and the Plus key to open Magnifier and zoom in.
Press Windows Minus to zoom out in Magnifier.
Press Windows Tab to open Task View.
Press Windows 1 through 9 to open a pinned taskbar app.
Press Windows Home to minimize all except active window.
Press Windows Semicolon to open emoji panel.
Press Alt Spacebar then X to maximize window.
Press Alt Spacebar then N to minimize window.
Press Alt Spacebar then R to restore window.
Press Alt Spacebar then C to close window.
Press Windows A to open Action Center or Quick Settings, depending on Windows version.
Press Windows M to minimize all windows.
Press Windows Shift M to restore minimized windows.
Press Windows U to open Accessibility settings or Ease of Access, depending on Windows version.
Press Windows Control D to create new virtual desktop.
Press Windows Control F4 to close current virtual desktop.
Press Windows Control Left Arrow to switch to previous desktop.
Press Windows Control Right Arrow to switch to next desktop.
Press Control Shift N to create a new folder in File Explorer.
Press Control Mouse Wheel to change icon size in File Explorer.
Press Alt Up Arrow to go up one folder level in File Explorer.
Press Shift F10 to open context menu for selected item.
Press Print Screen to copy a screenshot to the clipboard.
Press Control Escape to open Start menu.
Press Application key to open context menu.
Press F4 to display the address bar list in File Explorer.
